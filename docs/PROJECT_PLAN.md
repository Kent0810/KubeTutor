## Tech Stack

### Frontend
- **Next.js** - React framework for building the user interface
- **TypeScript** - Safer and more maintainable code
- **Tailwind CSS** - Fast and clean styling

### Backend
- **Next.js API Routes** - Simple backend for MVP
- **Node.js** - Server-side runtime

### Database
- **PostgreSQL** - Main relational database
- **Prisma ORM** - Easy database access and schema management

### Authentication
- **NextAuth.js** - User authentication and session management

### Learning Features
- **Markdown** - Store and render lesson content
- **React Hook Form** - Build quiz and admin forms
- **Zod** - Validation for forms and APIs

### DevOps
- **Docker** - Containerize the application
- **Docker Compose** - Local multi-service development
- **Kubernetes** - Future deployment and orchestration practice

### Tools
- **GitHub** - Version control
- **GitHub Actions** - CI/CD pipeline
- **Postman** - API testing
